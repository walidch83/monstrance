# Monstrance Clock — MIDI Practice Ritual

A browser-based practice tool for learning "Monstrance Clock" by Ghost on a MIDI keyboard —
no music theory, no scales, just phrase-by-phrase learning with real audio playback and
live MIDI feedback.

## Data source (important)

All 20 phrases in `song-data.js` are transcribed **programmatically from the actual MIDI file**
(`MonstranceClock.mid`), not guessed or OCR'd from the sheet music image. Every note, its exact
timing, and duration were extracted directly using the `mido` Python library, split into:
- Right hand = treble/melody track
- Left hand = bass/accompaniment track
- Phrases = 8-measure chunks across the full 152-measure song (bpm: 105, 4/4 time)

This means the note data is accurate and complete for the entire song, start to finish.

## How it works

1. Open the app in Chrome or Edge (Web MIDI + Web Audio work best there) with your MIDI
   keyboard connected via USB.
2. Pick a hand: Right → Left → Both.
3. Click "🔊 Play this phrase" — the app plays the phrase aloud using a built-in Web Audio
   piano synth AND highlights each key in gold as it's played, so you hear and see it first.
4. After playback finishes, the keyboard switches to practice mode: it highlights the first
   note(s) you need to play. Play them on your MIDI keyboard — correct notes flash green and
   the app auto-advances through the phrase.
5. Toggle "just messing around" to ignore velocity (great for slow practice); untick it once
   you want to play at real dynamics.
6. Click any phrase chip in "your rite so far" to jump directly to that phrase and hand.
7. Progress is saved automatically in your browser (localStorage) — close the tab and resume
   anytime, on the same browser/device.

## Files

- `index.html` — page structure
- `style.css` — Ghost-inspired dark ritual theme (candlelight, blood-red, bone-white)
- `app.js` — Web Audio piano engine, MIDI I/O, phrase/practice logic, progress persistence
- `song-data.js` — full song note data (generated from MonstranceClock.mid)
- `assets/midi/MonstranceClock.mid` — source MIDI file
- `assets/midi/MonstranceClock.pdf` — source sheet music (for reference)

## Regenerating song-data.js

If you replace the MIDI file, song-data.js can be regenerated by parsing it with `mido`,
grouping notes by track (right hand = higher average pitch track, left hand = lower),
chunking into 8-measure phrases, and exporting note/time/duration triples relative to each
phrase's start.

## Deploying

Static site, no build step. Push to GitHub, then import the repo into Vercel — it auto-detects
`vercel.json` and deploys as-is.
